# NGO Impact & Funding Insights Dashboard

This project presents an interactive Tableau dashboard analyzing NGO funding, beneficiaries, and campaign performance across cities.

## Key Insights
- Total donations: ₹8.36M
- 39,606 beneficiaries supported
- Program-wise funding distribution
- Monthly donation trends

## Features
- Interactive filters
- KPI indicators
- Program and city analysis
- Donation trend visualization

## Tools Used
- Tableau
- Data Visualization
- Dashboard Design
